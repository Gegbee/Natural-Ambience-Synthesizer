NATURAL AMBIENCE SYNTHESIZER

Steps to run this application yourself:

1. Install FFmpeg (Required)
2. Set Up a Virtual Environment (Optional, but recommended)
3. Install Requirements: pip install -r requirements.txt
4. Run: python app.py
